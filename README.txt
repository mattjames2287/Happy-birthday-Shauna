SHAUNA 30TH NFC GIFT SITE

UPLOAD STRUCTURE

Put this whole folder inside your GitHub Pages repo.

Recommended folder name:
shauna-30

Files:
shauna-30/index.html
shauna-30/style.css
shauna-30/assets/heres-to-thirty-shauna.mp3
shauna-30/assets/photo-1.jpg
shauna-30/assets/photo-2.jpg
shauna-30/assets/photo-3.jpg

WHAT TO REPLACE

1. Replace:
assets/heres-to-thirty-shauna.mp3

with the final MP3 from Suno.
Keep the same filename.

2. Replace:
assets/photo-1.jpg
assets/photo-2.jpg
assets/photo-3.jpg

with your own photos.
Keep the same filenames.

GITHUB PAGES URL FORMAT

If your repo is published at:
https://YOURUSERNAME.github.io/YOURREPO/

Then this gift page will be:
https://YOURUSERNAME.github.io/YOURREPO/shauna-30/

NFC TAG

Program the NFC tag with the final page URL:
https://YOURUSERNAME.github.io/YOURREPO/shauna-30/

IMPORTANT

Phones usually do NOT allow music to autoplay from a webpage.
The page will open and Shauna taps Play.
That is normal phone/browser security.
